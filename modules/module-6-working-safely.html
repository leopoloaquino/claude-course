Core workflowInspect first. Plan second. Edit third. Review fourth. This keeps work fast enough to be useful and controlled enough to be trustworthy.

Supporting habits

  1. Use the terminal without drama.
  2. Check the relevant files first.
  3. Review diffs before trusting changes.
  4. Prefer one small edit over many simultaneous ones.
